FTB-500 模組定位

模組用途
PM-500光功率模組，先量 dBm，判斷有沒有光 / 光弱不弱。
FTB-7200DOTDR 模組，找斷纖、接頭事件、彎折、距離。

光纖標準流程

1. ViewConn Pro 看端面：髒就清潔，刮傷就換跳線或重新端接。
2. PM-500 選正確波長：PON 下行 1490、上行 1310、CATV 1550、DWDM 看實際波長。
3. dBm 異常才上 FTB-7200D：短距離 Range 1~5 km、長距離 Range 10~50 km。
4. DWDM/CWDM 問題才上 FTB-5240S：看波長漂移、單波道功率低、OSNR 差。

RF / HFC 自動判斷

RF 工具分工

860 DSPi
Power / MER / BER / DOCSIS Lock，先量嚴重程度。
8591E
看 5~65 MHz 上行頻譜，確認 ingress、單頻干擾、底噪抬高。

ViewConn Pro + 清潔筆

  1. 接上待測跳線端面。
  2. 看畫面是否有灰塵、油污、刮傷。
  3. 髒污：清潔筆直插到底，按壓/旋轉一次。
  4. 再看一次端面，乾淨才准接到 PM / OTDR / OSA。

PM-500

  1. 在 FTB-500 進入 Power Meter。
  2. 選波長：1310 / 1490 / 1550 / 1625 依系統。
  3. 接入光纖，讀 dBm。
  4. -8~-20 正常；-20~-25 邊緣；<-25 光弱；<-30 近斷。

FTB-7200D OTDR

  1. 先清潔端面，接 Launch Fiber。
  2. 短距離：Range 1~5 km，Pulse 10~50 ns，Avg 15~30 秒。
  3. 長距離：Range 10~50 km，Pulse 100 ns~1 µs，Avg 30~60 秒。
  4. Start / Auto Test。
  5. 掉到底：斷纖；階梯：接頭/熔接損耗；局部凹：彎折。

FTB-5240S OSA

  1. 用在 DWDM/CWDM，不是 OTDR。
  2. 接入光輸出，選 Channel / Wavelength 範圍。
  3. 看每個波長功率、波長是否偏移、OSNR 是否差。
  4. 單一 Channel 低：查該波道發射端、MUX/DEMUX、跳線。

860 DSPi

  1. 接 Tap / 測試點 RF。
  2. 量 DS Power、MER/SNR、BER。
  3. 量 US Power / 回傳狀態。
  4. MER 低或 BER 有錯，進一步用 8591E 看頻譜。

8591E

  1. RF IN 前必要時加 10~20 dB Attenuator。
  2. 上行建議看 5~65 MHz。
  3. 多尖峰：Ingress；單頻高：設備干擾；整體抬高：線路污染/放大器問題。
  4. 從 Node / Tap 分段隔離追來源。
